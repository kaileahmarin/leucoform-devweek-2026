# Product vernacular

- **Leucoform:** the human-facing provenance and consent companion.
- **NoTUG:** the local governance engine, protocol, lifecycle, receipt system, and authority boundary.
- **Protected baseline:** the authoritative Git commit and tracked-file manifest bound at session start.
- **Managed session:** a disposable worktree in which an agent may propose changes.
- **Tug:** the frozen evidence package requesting authority for one exact proposal.
- **Grant:** a deliberate human action bound to one exact Tug. A Grant is not inferred from agent or
  process permissions.
- **Applied and verified:** the exact granted proposal was committed on a generated integration branch
  and its evidence was structurally verified.
- **Mutation Lock:** the default condition in which proposals have no integration authority.

NoTUG permits broad agent freedom inside its supported managed workspace while ensuring that no
high-risk, threatening, or unsafe proposal is implicitly granted without the user's permission.
